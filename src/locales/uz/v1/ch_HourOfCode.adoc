=== Ovoz Kliplarini Qo'shish

[role="curriculum-python"]
Kliplarning yuqorisida vaqt jadvali joylashgan. U soniyada vaqtni (yuqori qismi) va o'lchovni (pastgi qismi). O'lchov bu musiqada vaqt birligni bildiradigan atama.

[role="curriculum-python"]
Kliplar qatorlab ko'rsatilgan, va har bir qator trek deb nomlanadi.

[role="curriculum-python"]
image::../media/HOC/hoc_daw_labeled.png[]

[role="curriculum-python"]
{empty} +

[role="curriculum-python"]
Ovoz kliplarini trekka qoshadigan kodni kod muharririning 7-satrida joylashgan.

[role="curriculum-python"]
7-satr `fitMedia()` degan funktsiya qo'llanildi. Funktsia bu bir operatsiyani amalga oshiradigan bir maxsus kod.  `fitMeadia()` funktsiyasi vaqt jadvaliga bitta ovoz klipini qo'shadi.

[role="curriculum-python"]
Odatda funktsiya ishlashi uchun va aniq qanday ish bajarishni bilishi uhcun unga parametrlar talab qiladi. `fitMedia()` funktsiyasi o'ziga to'rtta parametir oladi:

[role="curriculum-python"]
* ovoz klipi
* trek raqami
* boshlang'ish o'lchov
* so'nggi o'lchov

[role="curriculum-python"]
Keling 7-satrni ko'rib chiqamiz:

[role="curriculum-python"]
[source,text]
----
fitMedia(RD_UK_HOUSE_MAINBEAT_8,1,1,5)
----

[role="curriculum-python"]
Bu degani "RD_UK_HOUSE_MAINBEAT_8" ovoz klipi 1-trekka, 1-o'lchovdan boshlab 5-o'lchovgacha qo'yiladi.

[[BeatStrings]]
=== Editing Beat Strings

[role="curriculum-python"]
EarSketch has a function called `reverseString()` which can reverse the order of the characters in a string.

[role="curriculum-python"]
On line 16 in the code, type the following:

[role="curriculum-python"]
[source,text]
----
fillRev = reverseString(fillA)
----

[role="curriculum-python"]
This creates a new variable called `fillRev` that contains a reversed version of `fillA`. Since `fillA` is `"0---0-0-00--0000"`, `fillRev` will be `"0000--00-0-0---0"`.
